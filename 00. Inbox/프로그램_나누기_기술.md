- 기능은 여러 작은 기능/로직(하위 기능)으로 구성


## 누니기는 역할/책임 도출
- 함수/클래스 분리 -> 역할/책임에 따라 구성 요소 도출

## 어떤 기준으로 나누나
- 답은 없다.
- 기능에서 "의미"있는 하위 기능 단위로 나누는 시도
	- 주문 취소 기능
		- 취소 사실 통지 vs PUSH 테이블에 INSERT
		- 결제 승인 취소 vs PG사가 제공하는 API 실행
