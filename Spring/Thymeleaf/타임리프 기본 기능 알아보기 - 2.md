## 연산

**주의사항**
><> <= >= 같은 비교 연산의 경우 주의해서 사용해야 한다.
>gt, lt, ge 같은 문자로 사용하는 경우 문자 그대로 출력이 될 수 있다.

기본적으로 사칙연산의 경우 다른 프로그래밍 언어와 똑같다.
Elvis 연산: 삼항연산자의 편의 버전
- #{data}?: false  -> 참이라면 데이터가 출력이 되고 거짓 이라는 false가 출력된다.
